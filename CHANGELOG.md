## [0.0.1] - 2020-11-25
### Added
- A class `Client` that wraps all documented API methods.
- End-to-end tests for the wrapper.
- A class `App` to run jobs over the checklists.
- Two basic jobs: `clean-urls` and `iconify-links`.
- A CLI to run the app.
